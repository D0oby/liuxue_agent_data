# Use Deterministic Unittest Coverage For Study Abroad RAG V1

Study Abroad Knowledge RAG V1 will be validated with the project's existing `unittest` style and a small deterministic fixture corpus rather than live sources or external embedding calls. The minimum suite covers manifest validation, Markdown/HTML/CSV/JSONL chunk locators, multilingual sparse tokenization, RRF hybrid behavior, query-intent ranking policy, anonymous internal privacy suppression, same-source saturation, and flat JSON formatting without leaking internal index objects.
