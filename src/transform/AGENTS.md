# AGENTS.md

`src/transform/` 负责把 Excel 原始字段转换成导入链路使用的结构化值。

## 目录职责

- 标准化课程名称。
- 解析学制区间。
- 解析开学季。
- 解析 IELTS 总分、最低小分和听说读写单项。
- 构建导入阶段 DTO 所需的结构化字段。

## 边界

- 不访问数据库。
- 不调用外部服务。
- 不做推荐评分或 REACH/MATCH/SAFETY 分档。
- 不根据 `CRICOS` 做唯一性假设。
- 不引入复杂 NLP；当前解析应保持确定性、可测试。

## 验证

修改解析规则时，优先补充精确输入输出测试，覆盖边界样例和不规则文本。
